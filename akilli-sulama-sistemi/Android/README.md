# 📱 Android Mobil Uygulama

## Özellikler

- Open-Meteo API ile 7 günlük hava durumu tahmini
- Bluetooth ile Arduino'ya veri gönderme
- Sensör verilerini gerçek zamanlı izleme
- Haftalık sulama planı görüntüleme

## Teknolojiler

- **Dil**: Kotlin
- **UI**: Jetpack Compose
- **Mimari**: MVVM
- **Bluetooth**: Android Bluetooth API

## Kurulum

1. Android Studio'da projeyi açın
2. Gradle sync yapın
3. Cihazınızda veya emülatörde çalıştırın
